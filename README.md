#LMS Front-End

### Setup instruction

1.Clone the project

```
    git clone https://github.com/KARTIKPAUL/lmsww

```

2.Move into directory

```
    cd lmsww

```

3.Install dependancies

```
    npm i

```

4.Run the server

```
    npm run dev

```

5.Set up instruction for Tailwind

```
   npm install -D tailwindcss postcss autoprefixer
   npx tailwindcss init -p
   
```
